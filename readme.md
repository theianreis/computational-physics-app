# Física Computacional 

Repositório dedicado à modelagem computacional de sistemas físicos, unindo teoria analítica, simulação numérica e visualização de dados. 

Esse repósitorio surge com a intenção de aplicar conceitos estudados na minha graduação em física em códigos funcionais e testáveis. 

Meu principal objetivo é para cada sistema físico existe um implementação computacional que permiti 

## O que o reposítorio cobre

- Mecânica Classíca:

- Termodinâmica 

- Astrofísica Extragaláctica

- Computação Quântica

- Análise de dados

- Outros tópicos serão adicionados conforme o andamento dos estudos e da pesquisa.

## Estrutura do repositório
 
```
computational-physics/
├── oscillators/        # osciladores harmônicos, amortecidos e forçados
├── mechanics/          # partículas, sistemas de N corpos
├── thermodynamics/      # processos termodinâmicos
├── notebooks/          # exploração e visualizações em Jupyter
├── tests/              # testes de validação (numérico vs. analítico)
└── docs/               # notas e referências
```

## Como Executar 
```bash
git clone https://github.com/theianreis/computational-physics-app.git
cd computational-physics-app
pip install -r requirements.txt
```
